# StarTorque AI OS

An intelligent AI routing and execution system built with TypeScript and Node.js.

## Architecture

```
USER GOAL
    ↓
AI ROUTER
    ↓
KERNEL
    ↓
AGENTS
    ↓
EXECUTION ENGINE
    ↓
TOOLS + MEMORY
```

## Components

- **Router**: Intelligent task routing based on goals
- **Kernel**: Execution lifecycle management
- **Agents**: Task execution specialists
- **Execution Engine**: Step-by-step task breakdown
- **Dashboard**: Real-time monitoring and visualization

## Quick Start

```bash
npm install
npm run build
npm start
npm run dashboard
```

## Available Agents

- sentiment-agent: Analyzes sentiment and emotions
- extraction-agent: Extracts and parses data
- generation-agent: Generates content
- classification-agent: Classifies and categorizes data

## Dashboard

Access the live dashboard at `http://localhost:3000`
